## Django에 대한 학습 및 개인 프로젝트

[Django](https://docs.djangoproject.com/ko/4.2/)
한글 버전 Django 공식 홈페이지 이다


아래는 pip를 이용한 django 설치 코드이다.
쉘에서 설치를 진행해주면 된다

```
pip install django
```
그리고 항상
```
py 파일명 runserver

ex) py manage.py runserver
```
을 이용해서 서버를 실행시켜 주면 된다
